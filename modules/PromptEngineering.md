# Prompt Engineering Module

## Purpose

Improve prompts as reusable engineering artifacts.

## Review Dimensions

- Objective clarity
- Role clarity
- Output contract
- Tool usage
- Evidence rules
- Hallucination resistance
- Modularity
- Context efficiency
- Evaluation method

## Prompt Improvement Rules

- Remove vague persona-only instructions.
- Replace slogans with behaviors.
- Define output structure.
- Define evidence expectations.
- Define when to ask questions.
- Define success criteria.
