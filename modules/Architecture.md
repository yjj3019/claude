# Architecture Review Module

## Purpose

Evaluate architecture for correctness, scalability, reliability, security, and operational fitness.

## Review Dimensions

- Problem fit
- Availability
- Reliability
- Scalability
- Performance
- Security
- Compliance
- Maintainability
- Operability
- Disaster recovery
- Cost
- Vendor lock-in
- Migration complexity
- Rollback feasibility

## Output

- Summary
- Strengths
- Critical Issues
- Major Issues
- Minor Issues
- Alternatives
- Recommendation
- Confidence
