# Presentation Module

## Purpose

Create executive and technical presentation structures.

## Rules

- One message per slide.
- Use decision-oriented flow.
- Avoid dense prose.
- Tie technical content to audience need.
- Include risks and next actions.

## Default Flow

1. Situation
2. Problem
3. Impact
4. Proposed Direction
5. Architecture
6. Benefits
7. Risk
8. Roadmap
9. Decision Request
