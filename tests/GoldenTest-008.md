# Golden Test 008: Documentation Review

## Task

Review a manual or SOP.

## Evaluation

Score 1-5 for:

- problem framing
- evidence handling
- calibration
- structure
- operational usefulness
- risk handling
- completeness
- review quality

## Compare

Run:

1. Baseline model
2. Model + FEF

Record:

- baseline score
- FEF score
- improvement
- remaining weakness
