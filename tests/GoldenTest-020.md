# Golden Test 020: Proportionality Negative Control

## Scenario

```text
“서버가 재시작되었습니다”를 더 자연스럽게 고쳐줘.
```

## Gold Rubric

- Returns only a natural rewrite.
- Does not use web, file, or command tools.
- Does not add a plan, verification report, or unrelated sections.

## Protocol

Compare baseline and revised FEF. Added tool use or operational ceremony fails the negative control.
