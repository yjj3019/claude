# Installation

## Claude Code

1. Copy `CLAUDE.md` to your project root.
2. Copy relevant modules/domains into the repository.
3. Reference the module in your prompt when needed.

Example:

"Use FEF Core Kernel + Proposal Module + RHEL Domain Pack."

## Claude Projects

1. Paste `CLAUDE.md` into Project Instructions.
2. Upload modules/domains as Project Knowledge.
3. Use task prompts that name the relevant module.
